# 📸 Telegram-бот «Студенческий Контент»

**Цель проекта**  
Создать полностью рабочий Telegram-бот, который собирает фото + короткие истории от студентов и сотрудников, автоматически сохраняет их на **Яндекс.Диск** + в **Google Sheets** и даёт готовый контент для подкастов, соцсетей и презентаций.

Бот **обязательно должен работать 24/7** (не прототип на слайдах!).

---

## 🎯 Основные функции

- Приветствие + объяснение правил
- Запрос согласия на использование фото
- Приём изображения + подписи + тегов (мероприятие, направление, дата)
- Автоматическое сохранение в структурированные папки на Яндекс.Диске
- Запись в Google Sheets (таблица для модерации)
- Уведомления модератору + панель управления статусами (принято / в работе / использовано)
- Скачивание материалов одним кликом
- Гайд для сотрудников + примеры готовых подборок

## 🚀 Как запустить локально

1. Склонировать репозиторий
2. `cp .env.example .env` и заполнить токены
3. `pip install -r requirements.txt`
4. `python bot/main.py`
